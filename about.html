Map all the things!

This is light weight web app that allows you to create markers on a map. Each marker gets notes and saves to your browser session. A database would be useful - then you can see other people's spots.

Next steps? A database(s) that will allow you to save your map. A public map which would let the crowd add their own spots. A custom/random color scheme? An ability to share your map by unique URL or password protected location. Aaaaand integrate Mountain Project, Trail Run Project, AllTrails, Weather, Food, whateva whateva. Provide suggestions if you'd like!
